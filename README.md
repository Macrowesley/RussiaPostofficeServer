## 在公司如何远程操作机器？
1. 机器在服务器的数据表中有记录
2. 服务器打开桌面frp快捷方式
3. 机器开机，登录服务器
4. 登录网页后台，点击ssh连接按钮
5. 机器可以成功远程升级
6. 关闭frp服务
